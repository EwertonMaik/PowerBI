# Power Query possui uma grande quantidade de conectores para importar e extrair dados de diversos tipo de arquivos e fonte de dados.

# Possui detecção automática do tipo de dados, e diversos tipos de dados para alteração e formatação.

# Possui funcionalidade de criar colunas personalizadas, através de valores de outras colunas.

# Possui funcionalidade de Substituição de valores por coluna ou por mais de uma coluna.

# Possui funcionalidade de duplicar uma coluna, dividir e extrair valores de uma coluna.

# Possui funcionalidade de extrair de um campo data as informações de Ano / Mês / Semana / Dia / Trimestre / Entre outros.

# Possui funcionalidade de excluír linhas em branco, duplicadas. Transformar Colunas em Linhas.

# Possui funcionalidade de preenchimento de linhas, com o valor da celula anterior.

# Possui funcionalidade de mesclar diversas tabelas em apenas uma.

# Possui funcionalidade de atualizar os parâmetros de caminhos das fontes de dados(Gerenciar Parâmetro). Podendo alteração manualmente em único local para todos, ou individualmente para cada arquivo. 

# Possui funcionalidade de importar uma pasta com todas as suas fontes de dados, não necessitando importar uma a uma. Muito bem utilizado para combinar os registros de diversas fontes em um único.

# Possui funcionalidade unir e criar uma tabela de fontes de dados diferentes, e depois mesclar colunas de outras fontes de dados na tabela criada.

# Possui funcionalidade de criar uma tabela a partir da outra por referência, diferente do caso criar uma tabela por duplica-la, tornando dois objetos independentes.

# Possui funcionalidade de importar uma arquivo .txt segmentado, onde em um único arquivo possui os registros de mais de uma tabela, e o Power Query possibilita ema importação e transformação, separando e criando cada tabela de uma única fonte de dados.

# Possui funcionalidade de criar uma coluna nova condicional, baseado em condições SE e SENÃO juntamente com operadores e valores.

# Possui funcionalidade que permite criar diretórios para agrupar e organizar as tabelas criadas e importadas das fontes de dados.

# Possui funcionalidade que identifica o relacionamento automático das chaves / tabelas. Podendo ser editadas e exluidas manualmente. O nome dos campos referente as chaves devem estar iguais para reconhecerem automaticamente. (Muito utilizado no relacionamento das tabelas FATOS e DIMENSÃO - Modelos STAR SCHEMA ou SNOW FLAKE)
